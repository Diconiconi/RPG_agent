# Source Notes

主要参考来源：

- BWiki《全npc流派一览》：用于确认四大家族、部分特殊 NPC 和各流派倾向。
- BWiki《特殊NPC的天道保送日期（更新中）》/《部分NPC保送时间》：用于确认四小强和部分重要 NPC 的 ID、擢升时间。
- BWiki 五大宗门页面：用于确认宗门掌门、长老和各自主修方向。
- BWiki《全支线攻略指南》与相关流程攻略：用于补充部分剧情定位。
- 3DM、18183 等攻略页：用于补充竹山宗高级修士和紫菱仙子相关资料。

注意：

- 本目录不是官方设定集。
- “身份定位”尽量基于公开资料。
- “性格底色、说话风格、重要动机”多为角色扮演用途的整理和推断。
- 未确认 NPC ID 的角色先不写 ID，后续可从游戏存档或配置表补齐。
