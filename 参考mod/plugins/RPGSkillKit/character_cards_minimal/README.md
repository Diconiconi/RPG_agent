# Minimal Character Cards

这里放“最小扩充方案”的 NPC 角色卡草稿。

这批文件不会被原 Mod 自动读取。真正要让原 Mod 生效时，需要把某个角色卡内容复制到游戏运行目录中的角色卡文件：

`C:\Users\22749\AppData\LocalLow\yusuiInc\觅长生\{npcId}_{npcName}_character.txt`

例如：

`20288_倪旭欣_character.txt`

当前格式只保留四块：

- 身份定位
- 性格底色
- 说话风格
- 重要动机

其中“性格底色”和“说话风格”有一部分是根据公开资料、剧情定位、功法流派和门派职能推断出的草稿，不是官方原文。后续你可以逐个角色微调。

目录：

- `families`: 四大家族相关角色。
- `sects/jinhong`: 金虹剑派。
- `sects/xinghe`: 星河剑派。
- `sects/lihuo`: 离火门。
- `sects/huachen`: 化尘教。
- `sects/zhushan`: 竹山宗。
- `special`: 主线、神兽、星宫、白帝楼等特殊角色。
- `peers`: 年轻一代、同辈竞争者和宗门剧情角色。
- `darkline`: 魔道、暗线、复仇线与灰色人物。
